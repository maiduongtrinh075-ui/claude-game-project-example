# Module Doc Reminder

## Trigger
When editing files under a module such as `src/gameplay/combat/` or `src/gameplay/inventory/`.

## Check
- did the module behavior change in a lasting way
- does `docs/systems/<module>.md` need an update
- does the module-local `CLAUDE.md` still match the current expansion phase

## Remind
- module docs should stay aligned with real behavior
- if phase ordering changed, update the module guide
