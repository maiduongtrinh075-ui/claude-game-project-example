# Pre-Commit Hook

## Trigger
Before creating a commit.

## Check
- review staged files for accidental scope creep
- confirm the task stayed within allowed scope
- check whether related docs should be updated
- run lightweight validation or tests when practical

## Remind
- do not commit unrelated file changes
- do not commit risky save or protocol changes without notes
- do not mark intermittent issues fixed without new telemetry
